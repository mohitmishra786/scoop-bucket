# scoop-bucket

Scoop bucket for tools by [mohitmishra786](https://github.com/mohitmishra786).

## Add this bucket

```powershell
scoop bucket add mohitmishra786 https://github.com/mohitmishra786/scoop-bucket
```

## Apps

| Name | Description | Install |
|------|-------------|---------|
| [agentop](https://github.com/mohitmishra786/agentop) | Terminal dashboard for AI coding assistant sessions | `scoop install mohitmishra786/agentop` |

## Usage

```powershell
# Add the bucket (one time)
scoop bucket add mohitmishra786 https://github.com/mohitmishra786/scoop-bucket

# Install any app
scoop install mohitmishra786/agentop

# Update all apps
scoop update *
```
